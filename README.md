# react-redux

This is a react-redux template, use react, redux, react-router, webpack, babel6

some examples:

1. counter
2. select area ( HOC )
3. drag ( by css )
4. dnd ( react-dnd )
5. transMotion ( react-motion )
6. sortable ( by drag )
7. draggableBalls ( react-motion )
8. donut ( d3 )

## Installation

```
npm install
```

## start

```
node .
```
